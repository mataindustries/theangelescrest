AC100 Crew & Pacers Logistics Plan

Short answer: Keep your crew plan simple: fixed meeting windows, one priority task list per aid station, and an offline backup if phones fail.

Last updated: 2026-02-18

Checklist

  • Build a one-car or two-car route plan before race week.
  • Assign one crew lead and one backup decision maker.
  • Pack aid-station bins by priority: fuel, feet, lights, weather layer.
  • Write fallback meet points for low-signal sections.

Steps

  1. Start with runner split estimates, then add generous arrival buffers.
  2. Define exactly what crew does at each stop in under three minutes.
  3. Stage pacer transitions with clear gear handoff rules.
  4. Rehearse your communication script so everyone gives the runner the same message.
